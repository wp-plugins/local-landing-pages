Reads the user-agent's HTTP_ACCEPT_LANGUAGE value and loads a page with a slug named after the language code.

== Description ==

Local Landing pages allows you to redirect the user on their first site visit based on their browser's default language. It uses the HTTP_ACCEPT_LANGUAGE header to determine your visitor's language. The menu in Settings > Local Landing Pages allows you to specify the path to a landing page for any particular language.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the paths to your local landing pages. Please note that all settings are updated automatically; there are no page loads.
4. Profit!
